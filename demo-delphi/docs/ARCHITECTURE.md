# demo-delphi

Nueva raíz Delphi 10.3 para replicar la arquitectura de `demo-sharp`.

Estado inicial:
- `src/Core`: contratos, enums, eventos y helpers base.
- `src/GUI`: shell principal compilable con `TMap`.
- `src/Symbology/LegendControl`: placeholder funcional de leyenda.

Pendiente por portar en siguientes etapas:
- `Databases`
- `ShapeEditor`
- forms avanzados de `Symbology`
